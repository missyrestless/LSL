#-----------------------------------------------------------------------------------+
#                       SimCaster (TM) YouTube TV Video Playlist                    |
#-----------------------------------------------------------------------------------+
# The playlist for the Easy YouTube TV.
#
# Format: 
#    Button Name|Youtube Video Name|YourTubeURL
#
#    Button Name and YouTube Video Name are optional
#    # - Comments and blank lines are ignored
#
#-----------------------------------------------------------------------------------
# A YouTube video playlist can have up to 50 videos, which will show up on the 
# "Playlist" menu.  More videos can be added, but they can only be played with 
# by choosing "Play All".  
#
# If a button name is omitted, the TV will call the playlist selections "Video 1",
# "Video 2", "Video 3", etc.  If a button name is omitted, then it is necessary to
# also leave out the YouTube Video name, and just supply the YouTube URL.
#
# Add as many playlist notecards to the TV as you wish.  Make each one a
# theme or collection.  You can swap between playlist notecards with the menu.
#-----------------------------------------------------------------------------------
#
# These are the sample playlists from the Easy Streaming TV and Spherical Theater.
# They illustrate the various ways a playlist entry can be formed including sample
# YouTube, Twitch, Daily Motion, and Vimeo entries. YouTube playlist URLs are alos
# supported as playlist entries. For examples of these see # Playlist_Playlists.txt
#-----------------------------------------------------------------------------------
