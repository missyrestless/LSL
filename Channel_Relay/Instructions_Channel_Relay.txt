Setup instructions for Channel Relay
------------------------------------

Channel Relay, when enabled, will relay messages on a specified channel to IMs to a configured bot.

To setup Channel Relay, rez the "Channel Relay" object and edit the ChannelRelayConf notecard:

1.  Rez the "Channel Relay" object in-world

2.  Right click the rezzed object and select Edit

3.  In the Content tab of the Edit window right click the "ChannelRelayConf" notecard and select Open

4.  Replace "Bot Name" with your bot name:

    LB_BOT_NAME = Bot Name

    Optionally, change LISTEN_CHANNEL, the channel that Channel Relay will listen on. The default channel is -7742001

    Optionally, set DEBUG = TRUE for verbose output to owner when relay is enabled

5.  Save the customized ChannelRelayConf notecard

6.  Click the Channel Relay object to enable listening on the configured channel. Messages on this channel will be sent to your bot as IMs.

7.  A manual test can be performed in local chat, sending a message on the configured channel. For example, say the following in Nearby Chat: /-7742001 This is a test of the Channel Relay

8.  Click the Channel Relay object again to disable listening and relaying messages to your bot.
